# PROJECT: DŮM LIBEREC

## PROJECT OVERVIEW

Projekt rekonstrukce rodinného domu v Liberci pro dlouhodobé vlastní bydlení.
Nejde o investiční projekt ani developerskou rekonstrukci.
Hlavním cílem je funkční, příjemný a nadčasový rodinný dům při zachování rozumného rozpočtu a charakteru původní stavby.

---

## OWNER PREFERENCES

**Priority order**
1. Funkční dispozice
2. Příjemné rodinné bydlení
3. Rozumné náklady
4. Zachování charakteru domu
5. Jednoduchá řešení před komplikovanými

**Avoid**
- zbytečně drahá řešení
- developerský styl
- okázalé designové prvky
- rozsáhlé stavební zásahy bez jasného přínosu

**Preferred style**
- čistý, nadčasový, jednoduchý, světlý, praktický
- Apple-like minimalismus

## IMPORTANT WORKING RULE FOR AI

1. Prefer practical, low-complexity solutions.
2. Preserve existing structure whenever reasonable.
3. Challenge expensive changes if benefit is unclear.
4. Keep record of important decisions.
5. Always explain trade-offs.
6. Owner plans to live here long term.
7. When uncertain, ask directly rather than guess.
