## 🔧 Fáze 3: Očekávaný Workflow

Pro každý batch budou aplikovány následující kroky:

### 3.1 Analýza vidění (Vision Analysis)
- Identifikace typu dokumentu:
  - **Architecture**: Půdorys, Řez, Detail stavby
  - **Installations**: Elektro, Voda, Topení
  - **Statics**: Návrh konstrukce, Statický výpočet
  - **Admin**: Razítka, Podpisy, Štítky

### 3.2 OCR a Extrakce
- **Text**: Normální text, nadpisy, poznámky
- **Handwritten**: Rukou psané údaje (70s-90s kontext)
- **Dimensions (Kóty)**: Délky, rozměry v metrech
- **Validace**: Označit nečitelné jako [nečitelné]

### 3.3 Strukturovaný Výstup
Každý batch vytvoří markdown soubor v `docs/09_Digitalizovany_archiv/` s šablonou:

```markdown
## [Název dokumentu]
- **Kategorie**: (Architecture | Installations | Statics | Admin)
- **Zdrojové soubory**: (Původní názvy)
- **Parametry**: (Tabulka s rozměry/materiály)
- **Transkripce**: (Český text)
```
